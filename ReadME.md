# Design Patterns

## Creational
Deals with class instantiation
- Abstract Factory
- Builder
- Factory Method
- Prototype
- Singleton

## Structural
Deals with how objects are composed and put together
- Adapter
- Bridge
- Composite
- Decorator
- Facade
- Flyweight
- Proxy


## Behavioural
Deals with how objects interact with one another
- Chain of Responsibility
- Command
- Interpreter
- Iterator
- Mediator
- Memento
- Observer
- State
